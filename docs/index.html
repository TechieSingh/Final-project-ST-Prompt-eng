Educational Content Generator

A generative AI project that creates educational content using RAG and prompt engineering.

About

This project generates educational materials like study guides, quizzes, explanations, summaries, and practice problems. It uses RAG to search through uploaded documents and prompt engineering to generate different types of content.

Features

  • RAG system with ChromaDB for document search
  • Prompt engineering for different content types
  • Web interface using Streamlit
  • Support for PDF and text document uploads
  • Multiple content types: study guides, quizzes, explanations, summaries, practice problems

Core Components

1. Retrieval-Augmented Generation (RAG)

  • Vector database (ChromaDB) for similarity search
  • Document chunking with overlap
  • OpenAI embeddings for semantic search
  • Top-k retrieval with filtering

2. Prompt Engineering

  • Specialized prompts for each content type
  • Context integration from RAG system
  • Edge case handling
  • Custom requirements support

Technology Stack

Python, Streamlit, LangChain, OpenAI API, ChromaDB

Setup

1. Install dependencies: pip install -r requirements.txt

2. Create .env file with your OpenAI API key

3. Run: streamlit run app.py

Performance

  • Content generation: 5-10 seconds
  • Retrieval time: 0.5-1 second
  • Content quality: 90%+ with good knowledge base

Ethical Considerations

  • All content is marked as AI-generated
  • Sources are tracked and can be cited
  • Basic content filtering implemented
  • Privacy considerations for user data
