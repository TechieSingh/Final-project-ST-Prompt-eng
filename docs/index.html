What It Does

This tool generates different types of educational content based on topics you provide. You can upload your own documents (PDFs or text files) to build a knowledge base, and the system will use that information to create more accurate and relevant content.

It can create:

  • Study guides with key concepts and examples
  • Quiz questions (multiple choice, true/false, short answer)
  • Detailed explanations of topics
  • Concise summaries
  • Practice problems with solutions

How It Works

The system has two main parts:

  • RAG System: Stores your documents in a vector database and finds relevant information when you ask questions
  • Prompt Engineering: Creates well-structured prompts for the AI to generate different types of content

When you request content, it searches through your uploaded documents (if you have any), combines that with the AI's knowledge, and generates the content you need.

Technical Details

Built with: Python, Streamlit for the web interface, LangChain for document processing, OpenAI's API for generation, and ChromaDB for storing document embeddings.

RAG Implementation: Documents are split into chunks, converted to embeddings, and stored in a vector database. When you ask for content, it finds the most relevant chunks and includes them in the prompt.

Prompt Engineering: Different prompts are used for each content type (study guide, quiz, etc.) to get the best results. The system also handles edge cases like empty inputs or overly long topics.

Getting Started

If you want to run this yourself:

  1. Clone the repository from GitHub
  2. Install dependencies: pip install -r requirements.txt
  3. Create a .env file and add your OpenAI API key: OPENAI_API_KEY=your_key_here
  4. Run the app: streamlit run app.py

That's it! The app should open in your browser.

Performance Notes

In my testing, content generation usually takes 5-10 seconds depending on the complexity. Document retrieval is pretty fast (under a second). The quality of the output depends a lot on having good source material in your knowledge base.

Important Notes

This is an AI-generated content tool, so keep in mind:

  • All content is AI-generated and should be reviewed for accuracy
  • The system tracks sources from your uploaded documents when available
  • There's basic filtering to prevent inappropriate content
  • No user data is stored permanently - everything is session-based
  • Make sure you have permission to use any documents you upload

Component Selection

This project implements two core components as required:

  • Prompt Engineering: Systematic prompting strategies with context management and specialized flows
  • RAG (Retrieval-Augmented Generation): Knowledge base with vector storage, document chunking, and retrieval mechanisms

These components were chosen because they work together effectively to create accurate, domain-specific educational content. RAG provides the knowledge grounding, while prompt engineering ensures high-quality, structured outputs for different content types.
